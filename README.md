\# 🩺 Predictive Modeling of Chronic Disease Trends



An Advanced Machine Learning Dashboard for Early Disease Detection.



\## 🚀 Project Overview

This project is an AI-driven healthcare tool designed to help medical professionals predict the risk of chronic diseases based on patient clinical data. Built using \*\*Python\*\* and \*\*Streamlit\*\*, it utilizes a \*\*Random Forest Classifier\*\* to provide real-time risk assessment.



\## ✨ Key Features

\- \*\*👤 Single Patient Assessment:\*\* Input clinical parameters (Age, BP, Cholesterol) via an interactive UI to get instant results.

\- \*\*📂 Bulk Data Analytics:\*\* Upload CSV files to process multiple patient records simultaneously for mass screening.

\- \*\*📊 Probability Scoring:\*\* Provides a "Risk Percentage" instead of a simple Yes/No, helping in clinical priority.

\- \*\*⚡ Cloud Deployment:\*\* Hosted on Streamlit Cloud for 24/7 remote accessibility.



\## 🛠️ Tech Stack

\- \*\*Language:\*\* Python 3.x

\- \*\*Libraries:\*\* Scikit-Learn, Pandas, NumPy, Streamlit

\- \*\*Model:\*\* Random Forest Algorithm

\- \*\*Deployment:\*\* GitHub \& Streamlit Community Cloud



\## 📋 How to Use

1\. Clone this repository.

2\. Install dependencies: `pip install -r requirements.txt`

3\. Run the app: `streamlit run app.py`



\## 📂 Dataset Source

The model is trained on the \*\*UCI Heart Disease Dataset\*\*, featuring 13 critical medical attributes.
